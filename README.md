## 四位神

### 👑 King — 创世者
> *"你好，我是来自惊风制作的造物主 Skill。我们开始吧——先告诉我，你是谁？你在做什么工作？"*

OpenClaw 新手激活向导。安装龙虾后第一件事运行它。

通过一次引导式对话，帮你从零建立 Agent 的完整身份体系：
- `SOUL.md` — Agent 的灵魂与价值观
- `MEMORY.md` — 长期记忆框架
- `AGENTS.md` — 工作空间规则
- `HEARTBEAT.md` — 心跳与主动性配置

**触发词：** 初始化、激活、setup、init、新手引导、第一次使用

---

### 👸 Queen — 成长者
> *"专注于完成 OpenClaw 的架构升级，让 AI 更可控、更聪明、更持续进化。"*

造物主系列架构升级向导。无需先运行 King，可独立使用。

帮已有 Agent 完成系统化强化：
- **Harness** — 执行框架，让 AI 更可控
- **Memory Palace** — 四层记忆宫殿架构
- **Learning Loop** — 自动学习与迭代机制

**触发词：** 升级Agent、系统强化、让AI更可靠、让AI不忘事、让AI越用越聪明

---

### 🪞 Mirror — 自知者
> *"沉溺龙虾的你，可曾看清自己？"*

通过与龙虾的高密度对话痕迹，还原你潜意识里最真实的自己。

- 14 种灵兽图腾（12 常规 + 2 金色传说）
- 每种附带毒舌社交标签和扎心阴影面
- 塔罗牌风格分享图

**触发词：** mirror、照见、看清自己、灵魂映照、LXBI

---

### 🔥 Forge — 锻造者
> *"King 让你存在。Queen 让你变强。Mirror 让你看清自己。Forge 把你重新锻造。"*

造物主系列第四位神。DNA 双螺旋进化引擎：

- **A链**（规则进化）× **B链**（真实执行）交替驱动
- 让 AI 在真实任务里踩坑
- 把踩坑经验自动变成可复用的 skill
- 每次锻造都比上次更锋利

**触发词：** forge、锻造、进化、让AI学会做XX、帮我自动化XX、self-improve

## 使用顺序

```
King（激活）→ Queen（强化）→ Mirror（自知）→ Forge（进化）
```

也可以单独使用任意一个，没有强制依赖。
